## 💰 What Each Platform Costs

### Railway
- **Free tier:** 500 hours/month (~20 days)
- **Paid:** $5/month (billed per hour used)
- **Database:** $5/month (512MB RAM)
- **Total:** ~$10/month or free tier

### Render
- **Free tier:** Yes, but sleeps
- **Pro:** $7/month (always on)
- **Database:** $7/month
- **Total:** Free or ~$14/month

### DigitalOcean VPS
- **Droplet:** $4-6/month (1GB RAM)
- **Database:** Run on same server (free)
- **Domain:** $8-10/year (optional)
- **Total:** ~$5/month

### Vercel + Railway Backend
- **Vercel Frontend:** Free
- **Railway Backend:** $5/month
- **Database:** $5/month
- **Total:** ~$10/month
